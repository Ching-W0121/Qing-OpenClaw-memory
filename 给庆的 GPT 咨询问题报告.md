## 🔄 我的操作流程

```python

browser(action="open", url="https://we.51job.com/pc/search?keyword=品牌策划&city=深圳")

# 2. 获取页面元素
browser(action="snapshot", refs="aria", labels=True)
# 返回：{"e190": "...", "e280": "...", ...}

# 3. 点击职位
browser(action="act", kind="click", ref="e190")

# 4. 确认进入详情页
browser(action="snapshot", refs="aria", labels=True, delayMs=3000)

# 5. 点击投递按钮
browser(action="act", kind="click", ref="e280")

# 6. 确认投递成功
browser(action="snapshot", refs="aria", labels=True, delayMs=3000)

# 7. 返回职位列表
browser(action="navigate", url="https://we.51job.com/pc/search?...")

# 8. 重复步骤 3-7（共 5 次）
```
