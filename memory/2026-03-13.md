## 🐛 飞书图片发送功能修复

### 问题
用户反馈飞书已开通图片上传权限，但无法发送图片到飞书。

### 根本原因
- `message` 工具的 `buffer` 参数（base64 data URI）没有被正确传递给飞书扩展
- 飞书扩展的 `sendMedia` 函数期望 `mediaBuffer`（Buffer 对象），但核心代码传递的是 `buffer`（字符串）
- 类型定义 `ChannelOutboundContext` 缺少 `mediaBuffer` 字段

### 修复方案
修改了两个文件：

1. **`types.adapters.d.ts`** - 添加类型定义
   ```typescript
   mediaBuffer?: Buffer;
   ```

2. **`outbound.ts`** - 修改 `sendMedia` 函数支持两种参数来源
   ```typescript
   // 支持 mediaBuffer（核心代码传递）和 buffer（message 工具传递）
   let effectiveMediaBuffer = mediaBuffer;
   const rawBuffer = (ctx as any).buffer;
   if (!effectiveMediaBuffer && rawBuffer) {
     const base64Data = rawBuffer.replace(/^data:[^;]+;base64,/, "");
     effectiveMediaBuffer = Buffer.from(base64Data, "base64");
   }
   ```

### 使用方法
```javascript
// 方式 1：base64 data URI
message(channel="feishu", message="图片", buffer="data:image/jpeg;base64,...")

// 方式 2：本地文件路径
message(channel="feishu", message="图片", mediaUrl="C:/path/to/image.jpg")
```
