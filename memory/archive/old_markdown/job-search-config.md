# BOSS 直聘岗位搜索配置

## 📋 匹配标准

### ✅ 核心条件

| 维度 | 要求 | 说明 |
|------|------|------|
| **职位关键词** | 品牌策划、品牌宣传、视觉设计、品牌设计、品牌视觉 | 标题必须包含其一 |
| **工作地点** | 深圳（排除宝安） | 宝安区域的岗位排除 |
| **工作经验** | 1-5 年 | 经验不限/应届 (薪资达标可接受)，5-10 年 (匹配度高可接受) |
| **学历要求** | 大专及以上 | 接受：大专、本科、学历不限、高中、中专、中技 |
| **目标行业** | 广告、策划、品牌咨询、营销服务、快消、文旅、互联网、科技、文化传播、传媒 | 优先这些行业 |
| **匹配度阈值** | ≥70% | 低于 70% 不推荐 |

### 🎯 岗位分类与薪资要求

**策划类岗位**
| 关键词 | 最低薪资 | 最优薪资 |
|--------|----------|----------|
| 品牌策划 | ≥8K | 10-15K |
| 品牌宣传 | ≥8K | 10-15K |

**设计类岗位**
| 关键词 | 最低薪资 | 最优薪资 |
|--------|----------|----------|
| 视觉设计 | ≥10K | 10-15K |
| 品牌设计 | ≥10K | 10-15K |
| 品牌视觉 | ≥10K | 10-15K |

### 🎯 薪资过滤逻辑

```
【策划类岗位】
薪资 < 8K → ❌ 直接排除
8K ≤ 薪资 ≤ 15K → ✅ 通过，正常计算匹配度
薪资 > 15K → ⚠️ 检查简历匹配度：
  - 经验要求与简历匹配 → ✅ 可推荐
  - 技能要求与简历匹配 → ✅ 可推荐
  - 否则 → ❌ 排除（可能要求过高）

【设计类岗位】
薪资 < 10K → ❌ 直接排除
10K ≤ 薪资 ≤ 15K → ✅ 通过，正常计算匹配度
薪资 > 15K → ⚠️ 检查简历匹配度：
  - 经验要求与简历匹配 → ✅ 可推荐
  - 技能要求与简历匹配 → ✅ 可推荐
  - 否则 → ❌ 排除（可能要求过高）
```

### 🎯 经验过滤逻辑

```
经验不限/应届 + 薪资达标 → ✅ 可推荐
1-5 年经验 → ✅ 正常通过
5-10 年经验 → ⚠️ 检查匹配度：
  - 匹配度 ≥ 80% → ✅ 可推荐
  - 匹配度 < 80% → ❌ 排除
10 年以上 → ❌ 直接排除（除非匹配度 ≥ 90%）
```

### 🎯 匹配度评分（满分 100%）

| 因素 | 权重 | 评分规则 |
|------|------|----------|
| 薪资匹配 | 30% | 最优范围内得 30 分，最低薪资边缘得 15 分，>15K 匹配得 25 分 |
| 经验匹配 | 25% | 1-5 年得 25 分，经验不限得 20 分，5-10 年匹配得 20 分 |
| 学历匹配 | 20% | 符合要求得 20 分 |
| 地点匹配 | 15% | 非宝安得 15 分 |
| 行业匹配 | 10% | 目标行业得 10 分 |

## 📝 去重机制

### 已推送记录文件
`memory/job-search-history.md`

### 去重规则
- 记录每个岗位的：公司名 + 职位名 + 首次推送日期
- 同一岗位 30 天内不重复推送
- 30 天后如岗位仍在招，可重新推送

### 记录格式
```markdown
| 推送日期 | 公司 | 职位 | 薪资 | 链接 | 状态 |

```
