## 🚨 BOSS 直聘封号事故

**时间：** 2026-03-07 22:30  
**封禁原因：** 高频抓取触发风控  
**封禁时长：** 约 5.5 小时（至 04:09）

### 错误行为总结
1. **高频快照** — waitForPageLoad 中每秒调用一次 browser.snapshot()
2. **单一操作路径** — 直接导航深层 URL，未模拟首页搜索流程
3. **廉价延迟** — 3-8 秒随机延迟无意义，无鼠标轨迹模拟
4. **过度清理缓存** — 每次 clearCookies，装成"新用户"
5. **无 IP 质量检测** — 在灰名单 IP 上继续操作

### 教训
- **稳定输出高于一切** — 不要为了完成任务而鲁莽
- **模拟真实用户行为** — 从首页开始，搜索框输入，滚动浏览
- **熔断机制** — 3 次失败自动停止，不暴力重试
- **限流保护** — 10 次/小时搜索上限，不掐点执行
