## 📋 匹配逻辑（来自 匹配逻辑.docx）

## ⚠️ 遇到的问题及解决方案

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| GitHub 连接失败 | 网络不稳定 | 手动下载 ZIP 安装技能 |
| 浏览器自动化被检测 | BOSS 直聘反爬严格 | 改用 Tampermonkey 客户端脚本 |
| CSS 选择器不匹配 | 网站结构变化 | 多选择器轮询 + 文本解析 |
| 跨标签页操作限制 | 浏览器同源策略 | postMessage 通信 + 手动 fallback |
| 页面持续刷新 | 反爬机制触发 | 降低频率，增加延迟 |
