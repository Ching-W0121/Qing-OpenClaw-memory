## 🎯 核心原则

### 记忆的第一原则

> **记录"用户意图"，而不是"系统行为"**

**错误做法**（机器视角）:
- Git commit
- 执行日志
- API 调用
- 技术任务

**正确做法**（用户视角）:
- 用户问过什么
- AI 帮用户做过什么
- AI 答应过用户什么
- 双方讨论过什么

**公式**:
```
用户记忆 = 用户请求 + AI 回应 + 最终结果
```

---

## 📋 三层记忆结构

### 1. 用户层（最重要）

记录用户的请求/意图

**示例**:
```
2026-03-03
用户请求：查找尼日利亚电商平台 Jumia 的流量情况
```

或
```
用户请求：帮我写 Facebook 广告文案
```

**这是未来可回忆的核心数据**

---

### 2. 任务层

记录 AI 执行的任务

**示例**:
```
任务：分析 Jumia 市场流量
执行内容：
- 搜索平台数据
- 分析竞品
- 生成报告
```

---

### 3. 技术层（最不重要）

- Git commit
- 日志
- API 调用

**只对开发者有用**

---

## 📝 请求追踪设计

### Request Log 结构

```
RequestID | 时间 | 用户请求 | 任务类型 | 结果摘要 | 状态
```

**示例**:
```
ID: R20260303-01
时间：2026-03-03
用户请求：查尼日利亚电商市场
类型：市场研究
结果：提供 Jumia、Konga 平台分析
状态：完成
```

### 检索能力

当用户问："上周我让你查了什么？"

系统执行：
```sql
SELECT requests WHERE date BETWEEN last_week
```

返回：
```
上周你让我做了 3 件事：
1. 查尼日利亚电商市场
2. 写 Facebook 广告文案
3. 分析净水器产品定价
```

---

## 💬 对话记录 ≠ 记忆

**错误**: 存储原始对话
```
用户：帮我查尼日利亚电商
AI：好的
AI：这里是分析...
```

**正确**: 摘要后存储
```
用户请求：尼日利亚电商市场研究
AI 输出：Jumia 市场报告
```

**流程**: 对话 → 摘要 → 记忆

---

## 🤔 反思机制设计

### 三种反思机制

#### 1. 任务反思（最重要）

每个任务结束时生成任务总结：
```
任务：尼日利亚市场分析
总结：
- 用户目标：进入非洲电商市场
- 提供内容：Jumia 市场数据
- 可能下一步：寻找物流方案
```

#### 2. 周反思

每周自动总结：
```
本周用户主要关注：
1. 非洲电商
2. Facebook 广告
3. 净水装置产品
```

#### 3. 长期画像

形成用户兴趣画像：
```
用户长期关注：
- 跨境电商
- 非洲市场
- 产品营销
```

### 可见性原则

> **默认不可见，但可被用户询问**

用户问："你对我有什么观察？" → AI 才展示反思

---

## 🤖 自动记录 vs 手动记录

### 自动记录（三类）

#### 1. 用户明确请求
- "帮我查..."
- "帮我分析..."
- "帮我写..."

#### 2. 重要决策
- "决定进入尼日利亚市场"

#### 3. 长期信息
- "用户做跨境电商"

### 不记录（避免过载）

- 闲聊
- 表情
- 小问题
- 无结果对话

**经验值**: 1000 条对话 → 只需要 30 条记忆

---

## 🗂️ 记忆结构最佳实践

### 四类记忆

#### 1. 用户画像（长期稳定）
- 职业
- 兴趣
- 目标

#### 2. 请求历史（可查询）
- 用户让我做过什么

#### 3. 项目记忆（上下文）
- 非洲电商项目
  - 市场研究
  - 产品方案
  - 物流方案

#### 4. 关系记忆（人际关系）
- 用户认识的人：素素

## 🔍 检索系统设计

### 用户查询方式

**错误**: `search request log week=1`  
**正确**: "上周我让你做什么"

### 检索流程

```
1. 解析问题
2. 识别时间范围
3. 搜索请求记录
4. 返回摘要
```

### 示例

用户："我最近研究什么？"

系统：
```
最近你主要在研究：
1. 非洲电商
2. 净水装置销售
3. Facebook 广告
```

## 🎯 终极设计原则

> **记忆系统只有一个核心目标：帮助 AI 回答用户的"过去问题"**

用户真正的问题永远是：
1. 我之前说过什么？
2. 你帮我做过什么？
3. 我们进行到哪一步了？

**如果能回答这三件事，记忆系统就成功了**
