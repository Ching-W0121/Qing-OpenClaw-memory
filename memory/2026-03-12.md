## 📝 GPT 老师教学总结

### 完整 AI Memory 架构

```
conversation → episodic → distillation → semantic + vector → reasoning
                                            ↓
                                      reflection → strategy
```

### Memory Reflection Loop

- **Distillation**：从对话提取事实 → user_profile / facts
- **Reflection**：从经历总结经验 → strategy / lessons

### 关键设计原则

1. 批量蒸馏（每 50 条或 6 小时），不是每条对话都做
2. 异步写入，不阻塞主流程
3. Crash protection：启动时检查 JSONL 完整性
4. 结构化输出：JSON 格式便于后续处理
