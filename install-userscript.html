🤖 BOSS 直聘自动投递脚本

版本：1.0 | 用户：王庆

📦 步骤 1：安装脚本

点击下方按钮，Tampermonkey 会自动识别并打开安装页面：

或者手动操作：

  1. 点击浏览器右上角的 Tampermonkey 图标（🧩）
  2. 点击 "添加新脚本"
  3. 删除所有模板代码
  4. 复制下方文本框中的完整代码
  5. 粘贴到编辑器中
  6. 按 Ctrl+S 保存

🚀 步骤 2：开始投递

  1. 打开 BOSS 直聘 - 品牌策划职位列表
  2. 确保已登录账号
  3. 页面右下角会出现 "🤖 BOSS 自动投递" 控制面板
  4. 点击 "▶ 开始" 按钮
  5. 脚本会自动筛选并投递匹配的职位

⚙️ 配置说明

  • 薪资范围：8K-15K
  • 排除地区：宝安
  • 学历要求：大专/本科/不限
  • 匹配度：≥70% 才投递
  • 投递间隔：15 秒/职位
  • 最大投递：30 个/次
