## 五、组件架构

### 5.1 防封禁组件层次

```
┌─────────────────────────────────────────┐
│         应用层（Adapter/Pipeline）        │
├─────────────────────────────────────────┤
│  CircuitBreaker  │  OperationLimiter   │
│  (熔断保护)      │  (限流保护)          │
├─────────────────────────────────────────┤
│      SessionManager (会话持久化)         │
├─────────────────────────────────────────┤
│         存储层（文件/内存缓存）           │
└─────────────────────────────────────────┘
```

### 5.2 调用流程

```
1. 开始工作
   ↓
2. SessionManager.get_or_create_session()
   ↓
3. OperationLimiter.can_proceed(SEARCH)
   ↓
4. CircuitBreaker.call_async(search_func)
   ↓
5. 执行搜索
   ↓
6. 成功→CircuitBreaker._on_success()
   失败→CircuitBreaker._on_failure()
   ↓
7. OperationLimiter.record(SEARCH)
   ↓
8. SessionManager.record_activity()
   ↓
9. 结束工作→OperationLimiter.end_work_session()
```
