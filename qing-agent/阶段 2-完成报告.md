## 五、适配器对比

### 5.1 BOSS vs 智联

| 特性 | BOSS 直聘 | 智联招聘 |
|------|----------|----------|
| **搜索 URL** | `/web/geek/job?city={code}&query={kw}` | `/job/positions?keyword={kw}&city={code}` |
| **城市代码** | 101280600（深圳） | 530（深圳） |
| **页面结构** | 职位卡片 class 含"job-card" | 职位列表 class 含"joblist" |
| **详情页面** | 需要登录 | 部分公开 |
| **投递方式** | 立即沟通 | 立即申请 |
| **数据质量** | 高 | 中 |

### 5.2 适配器架构

```
platform/
├── base_adapter.py       # 统一接口
├── boss_adapter.py       # BOSS 直聘实现
├── zhilian_adapter.py    # 智联招聘实现
└── adapter_factory.py    # 工厂模式
```

**核心优势：**
- 统一接口，易扩展
- 服务层无需修改
- 支持多平台搜索
