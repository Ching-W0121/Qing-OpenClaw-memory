### 2. 已更新的方法

#### `parse_job_detail(snapshot)`
**更新内容**:
- 支持 heading[level=1] 提取职位标题
- 支持 text 元素提取薪资
- 支持 listitem 提取地点/经验/学历/类型/招聘人数
- 支持 generic 提取技能标签
- 支持 text 元素（数字开头）提取职位描述
- 支持 button 提取行业信息
- 支持 heading[level=3] 提取 HR 信息
- 支持 generic 提取 HR 活跃状态
- 支持 text 提取详细工作地点

**新增字段**:
- `salary_min` / `salary_max` - 标准化薪资范围
- `location` - 详细工作地点
- `recruit_count` - 招聘人数
- `job_type` - 工作类型（全职/兼职）
- `hr_name` - HR 姓名
- `hr_title` - HR 职位
- `hr_status` - HR 活跃状态
