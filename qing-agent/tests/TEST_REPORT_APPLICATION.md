## 📝 注意事项

### 投递前准备
1. **登录状态** - 确保浏览器已登录目标平台
2. **简历完整** - 确保平台简历已填写完整
3. **验证处理** - 拉勾网可能需要人工验证

### 投递限制
- **智联招聘**: 每日投递次数限制
- **前程无忧**: 部分职位需要附件简历
- **拉勾网**: 需要在线简历
- **猎聘网**: 高端职位可能需要定制简历
