**原因分析**:
- 拉勾网有反自动化检测
- 未登录状态 + 自动化访问 = 触发验证

**解决方案**:
1. **人工验证后使用** - 先手动通过验证，然后让 Agent 继续操作
2. **登录状态** - 登录后可能减少验证频率
3. **使用前程无忧替代** - 前程无忧更友好

**结论**: **拉勾网未登录状态受限，需要人工验证或登录后使用。**
