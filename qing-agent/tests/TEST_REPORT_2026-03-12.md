## 🛡️ 保护机制说明

### 3.1 重试机制 (Retry Limit)
```python
max_retry = 3
```
- 每次失败自动重试
- 最多重试 3 次
- 超过 3 次后触发熔断器

### 3.2 超时保护 (Timeout)
```python
timeout_seconds = 60
```
- 每个操作最多 60 秒
- 超时后抛出 `asyncio.TimeoutError`
- 计入失败次数

### 3.3 熔断器 (Circuit Breaker)
```python
circuit_breaker_threshold = 3      # 连续失败 3 次触发
circuit_breaker_cooldown = 1800    # 冷却 30 分钟
```
- 连续失败 3 次后打开熔断器
- 熔断器打开期间拒绝所有请求
- 30 分钟后自动关闭（冷却）
